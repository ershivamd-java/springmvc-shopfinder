# 🛍️ Spring MVC ShopFinder

A web application developed using **Spring MVC, Hibernate, JSP, MySQL, and Maven** for managing and searching shops.

---

## 🚀 Technologies Used

- Java
- Spring MVC
- Hibernate
- JSP
- JSTL
- MySQL
- Maven
- HTML5
- CSS3
- Bootstrap
- JavaScript
- Apache Tomcat

---

## ✨ Features

- User Registration
- User Login
- Forgot Password
- Shop Registration
- Search Shops
- Product Management
- Review System
- Responsive UI

---

## 🛠️ Tools Used

- Eclipse IDE
- MySQL Workbench
- Apache Tomcat
- Git
- GitHub

---

## 📂 Project Structure

src/
src/main/java
src/main/resources
src/main/webapp
pom.xml

---

## ⚙️ How to Run

1. Clone the repository
2. Import as Maven Project in Eclipse
3. Configure Apache Tomcat
4. Create MySQL Database
5. Update database username & password
6. Run on Server
